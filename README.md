# Docker-yii2-nextjs

Yii2, NextJS, Crontab, NodeJS. All in one docker compose container.

## preinstall
```
docker run  -v %cd%:/app -it --rm composer install -d yii2 --prefer-dist
# sudo docker
docker exec -u 0 -it containerName bash
```
## run production
```
buld-run-dev.bat
```
## run development
```
buld-run-prod.bat
```
